# 致谢

这本书的想法源于 2020 年 4 月的一个笔记。我问我的妻子 Kholoud："你觉得写一本关于调试 CSS 的书怎么样？"我告诉她这将是一本很短的书（最多 60 页）。七个月后，这本书已经达到了 300 页。Kholoud 是第一个支持这个想法的人，她坚持认为我应该继续下去，现在我们终于完成了。谢谢你，我最亲爱的人！

来自社区的第一个鼓励我的人是 John Allsopp 先生。他邀请我在 Web Directions 会议上分享这本书的主题，并且是最早的支持者之一。非常感谢！

我要感谢 Geoffrey Crofte。他非常热心，友善地校对了整本书，并指出了许多需要修改的地方。非常感谢！

最后，我想感谢 Bram Van Damme，他审阅了这本书的初稿。他指出了我应该改进的一些重要事项。谢谢你，Bram！
